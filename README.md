# Connect Four Game

A React-based Connect Four game built with Vite.

## Installation

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm run dev
```

The game will open automatically in your browser at `http://localhost:5173`

## How to Play

- Players take turns clicking on the column buttons to drop their piece
- Red pieces (X) go first, Orange pieces (O) go second
- First player to get 4 in a row (horizontally, vertically, or diagonally) wins
- Click "Reset Game" to start a new game

## Build

```bash
npm run build
```

## Features

- Real-time game state management
- Automatic winner detection
- Responsive UI
- Simple and intuitive controls
