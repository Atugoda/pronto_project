Two player game
In a two-player game, players take alternate turns. They have been given particular colour tokens which they must
drop into a grid. The grid has 7 vertical columns where each has 6 slots. The player who forms a line of 4 tokens 
(vertical. Horizontal or diagonal) first will win the game. 
Coding Challenge – Implement simulation of this game using ReactJS.
Following are the requirements. Complete as many as possible.
• Implement the UI to show the game board’s status.
• Indicate which player’s turn – (Red/Yellow)
• Clicking on a button at the top of a column will fill the next available slot with current player’s token colour.
• Indicate if there is a winner by connecting four tokens horizontally, vertically or diagonally
